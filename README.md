# 3D Coverflow – One Page Portfolio

A responsive, single-page static site showcasing a 3D coverflow image gallery with smooth scrolling navigation and polished sections (Home, About, Cats, Contact). Built with HTML, CSS, and vanilla JavaScript—no build step or dependencies required.

## How to open locally
- Option 1: Open `index.html` directly in your browser (double-click or drag into a tab).
- Option 2: Serve with a simple local server for best results:
  - Python 3: `python3 -m http.server` then visit `http://localhost:8000`
  - Node (npm): `npx serve .` then visit the printed URL

## Sections
- Home: 3D coverflow gallery with autoplay, keyboard, touch, and dot navigation.
- About: Overview of the template’s features and design approach.
- Cats: New article section describing the merits of having cats as pets, including independence, emotional wellness, suitability for small spaces, cleanliness/litter training, cost considerations, quiet presence, bonus pest control, enrichment and responsible ownership, and notes on allergies and lifestyle fit.
- Contact: Contact information, social links, and a basic message form.
