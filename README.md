# 3D Coverflow — One Page Portfolio

This is a single-page, responsive portfolio template featuring a 3D image coverflow, an About section, a new Cats essay section, and a Contact section.

## New: Cats Essay Section
We added a well-written content section that explores the merits of having cats as pets. It includes:
- An H2 title and intro
- A list of reasons cats make wonderful companions
- Responsible ownership considerations
- A warm closing paragraph

You can access it from the top navigation via the “Cats” menu item, or by visiting the anchor `#cats`.

## How to View
Simply open the `index.html` file in your web browser:
- Double-click `index.html`, or
- Drag `index.html` into a browser window

No build tools or servers are required.
