# 3D Coverflow — One Page Portfolio

This is a static, single-page portfolio template that showcases a 3D image coverflow with smooth interactions and responsive design.

## What's Included

- Home section with 3D coverflow gallery
- About section describing the template
- Cats article section: “The Merits of Having Cats as Pets”
- Contact section with information and a form UI

## Cats Article Section

This template now includes an article titled “The Merits of Having Cats as Pets,” accessible from the top navigation (Cats). You can edit the content directly in:
- index.html, under the section with id="cats"

The article is structured with semantic headings, lists, and uses existing site classes for consistent styling.

## Local Preview

Because this is a static site, you can view it locally without a build step:
1. Clone or download this repository.
2. Open index.html in your web browser (double-click the file or drag it into a browser window).

No additional dependencies or servers are required.
